# SondirPro — Landing Page

Ini adalah landing page sederhana untuk jasa sondir (pengujian daya dukung tanah).

Cara menjalankan (paling sederhana):

- Buka langsung `index.html` di browser Anda.

Atau jalankan server lokal (direkomendasikan saat menggunakan modul JS atau testing CORS):

PowerShell (Windows):
```
# jika memiliki Python 3
python -m http.server 8000; # lalu buka http://localhost:8000
```

Berisi file:

- `index.html` — halaman landing
- `styles.css` — stylesheet
- `script.js` — validasi form & perilaku kecil

Ganti gambar placeholder di `index.html` dengan aset Anda sendiri jika perlu.
